Flutter package for loading image with shimmer and fade in effect for smooth transition

## Features

[![Screen shot](./assets/screenshot.gif)](./assets/screenshot.gif)

## Getting started

```yaml
dependencies:
  shimmer_image: ^0.0.1
```

## Usage
```dart
ProgressiveImage(
              width: 300.0,
              image: 'https://wallpapercave.com/wp/wp4600617.jpg',
              height: 200.0,
            ),
```
